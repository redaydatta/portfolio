Web Application

Easy Merchant App

A full-featured e-commerce platform built with Laravel and MySQL, featuring a seamless shopping experience with secure payment gateway integration (Stripe & PayPal), real-time inventory management, order tracking, and a powerful admin dashboard with analytics.

Technologies Used
Laravel MySQL Bootstrap Stripe API Redis Vue.js
Key Features
  • Multi-vendor product catalog with search & filtering
  • Secure payment integration (Stripe & PayPal)
  • Real-time inventory & order management system
  • Admin dashboard with sales analytics & charts
  • Customer review & rating system
  • Email notification & invoice generation
