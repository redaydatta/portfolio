Task Management App

A real-time collaborative task management application built with React and Node.js. Teams can create boards, assign tasks, track progress with Kanban-style workflows, and communicate through in-app messaging — all synced live via WebSockets.

Technologies Used
React Node.js MongoDB Socket.io JWT Tailwind CSS
Key Features
  • Kanban-style drag-and-drop task boards
  • Real-time collaboration with WebSocket sync
  • Team member roles and permission management
  • Progress tracking with visual Gantt charts
  • In-app messaging and file attachments
  • Email & push notification system
